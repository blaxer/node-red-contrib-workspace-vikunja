# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-06-07

### Added

- Collapsible task descriptions with HTML rendering support
- Click task title to expand/collapse description
- "Show on all flows" option to display tasks on every flow
- Smooth animations for collapsible elements
- HTML rendering for task descriptions (lists, paragraphs, etc.)

### Changed

- Updated task display to use modern collapsible element approach
- Improved code organization for maintainability
- Enhanced documentation with new features

## [1.0.0] - 2026-06-06

### Added

- Initial release
- Display Vikunja tasks on Node-RED workspace
- Toggle task completion status
- Add new tasks
- Edit existing tasks
- Delete tasks
- Auto-refresh support
- Drag task container to reposition
- Collapsible task list header
