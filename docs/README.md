# Node-RED Vikunja Tasks

A Node-RED node to display and manage Vikunja tasks on the workspace canvas.

## Features

- Display tasks from Vikunja on the Node-RED workspace
- Toggle task completion status
- Add, edit, and delete tasks
- Collapsible task descriptions with HTML rendering
- Show on all flows option
- Auto-refresh support

## Installation

```bash
npm install node-red-contrib-workspace-vikunja
```

## Configuration

- URL: Your Vikunja server URL
- Project ID: Project to display tasks from
- API Token: Your Vikunja API token
- Show Completed: Toggle completed tasks
- Show on all flows: Display on all flows
- Refresh: Auto-refresh interval (minutes)
